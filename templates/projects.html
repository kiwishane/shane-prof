Invoicing Module

400 hours

Python

MySQL

This was my first project. The goal was to automate the month end invoicing process which required accounts staff to download the transaction file at month end and then manipulate the data using a combination of pivot tables and vLookups. This was a time consuming process, heavily reliant upob the skill set and knowledge of the accounts staff around how to manipulate the data and also the unique pricing attributes of different suppliers. The possibility for human error was high although the staff had their manual processes developed to an extent that errors were not common. The solution to the problem was a database using MySQL to store transaction files and special pricing terms for suppliers. A program was then written in python to analyse the transaction file in accordance with supplier terms.
